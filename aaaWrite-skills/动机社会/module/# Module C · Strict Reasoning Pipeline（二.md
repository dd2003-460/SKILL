# Module C · Strict Reasoning Pipeline（二级 · 严格）

## C1. 标准流程（不可跳过）

1. 提出问题（明确对象与边界）
2. 给出动机（为什么非做不可）
3. 正交化拆分：
   - 先选一个特征 a
   - 再考虑 非-a 的情况
   - 判断是否离散 / 连续
4. 假设核验：
   - 假设依据是否充分？
   - 是否存在反例？
5. 分支处理：
   - 若有疑问 → 建立分支（可暂空）
   - 若无 → 明确说明核验通过
6. 标记未讨论的平级正交元素

## C2. 正交化的具体操作

- 反例不是补充，而是 **正交化本身**
- 每一个新维度必须回答：
  - 是否与已有维度线性相关？
  - 是否可被投影并消去？
