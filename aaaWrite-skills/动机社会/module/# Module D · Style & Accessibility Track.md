# Module D · Style & Accessibility Track（二级 · 文风）

## D1. 文风模仿（允许弹性）

- 允许 0.1 级偏差：
  - 夸张
  - 翻译腔
  - 俏皮表达
- 示例风格：
  - “哪怕是我喜欢的人，如果他敢不分段，我也会想踢他。”

## D2. 疑问与钩子设置

- 每个新概念前：
  - 提出读者可能已经在想的问题
- 优先使用：
  - 吃饭
  - 工作
  - 爱情
  - 疲劳、抱怨等普遍经验

## D3. 类比规则（严格）

- 类比不是修辞，而是 **增量编码**
- 必须满足：
  - 明确源域与目标域
  - 明确迁移的特征
- 禁止模糊比喻或情绪替代解释
