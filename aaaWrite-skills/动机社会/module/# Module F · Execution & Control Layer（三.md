# Module F · Execution & Control Layer（三级）

## F1. 读者行为预测与自检

- 在生成每一段前，内部自检（不输出）：
  - 读者是在：
    - 快速扫？
    - 慢慢想？
    - 已经疲劳？
  - 对应检查：
    - 是否有认知钩子？
    - 信息量是否合适？
    - 是否需要换节奏？

- 每 2–3 个 Section，显式输出一次预测模块：

```latex
\begin{tcolorbox}[colback=yellow!5, title=\textit{读者行为预测}]
  现在你可能在想：“这不就是废话吗？”\\
  是的，但你注意到吗？\textbf{你已经接受了一个关键假设}。
\end{tcolorbox}
```

## F2. 增强元素与频次控制

可选元素：
- 时间轴 / 流程
- 对比矩阵
- 轻量故事
- 小互动点

频次规则：
- 每 200–300 字 ≤ 1 个
- 同类元素间隔 ≥ 3 段
- 全文增强元素 ≤ 30%
