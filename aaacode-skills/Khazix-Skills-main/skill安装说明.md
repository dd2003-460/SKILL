# Skill安装说明

## 概述
本仓库包含多个自定义技能（Skills），用于增强AI模型的行为和功能。每个技能文件夹下包含SKILL.md（技能定义）和README.md（使用说明）。

## 技能列表
1. ralph-wiggum-skill：自主迭代技能，让AI自己修改代码。
2. planning-with-files-skill：用Markdown文件进行任务规划。
3. Superpowers-skill：技能库的技能库，升级规划模式。

## 安装方法
1. 复制SKILL.md的内容到AI工具的系统提示中。
2. 参考README.md获取使用建议。
3. 对于批量加载，使用skill-loader工具合并所有技能。

## 注意事项
- 确保AI工具支持系统提示。
- 测试时从简单任务开始。
