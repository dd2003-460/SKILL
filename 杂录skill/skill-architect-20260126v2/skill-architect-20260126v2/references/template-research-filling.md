# Research Template - Filling Guide

## Summary Section
- Directly answer the user's research question, avoid preamble
- Keep it concise (3-4 sentences)
- Write in Chinese, but keep technical terms in English

## Detailed Findings
- Group by functional module or file
- Each group must include: file path, line numbers, findings, gaps
- Write descriptions in Chinese
- Use concrete code references in English (e.g., `path/to/file.py:123-145`)

## Component Connections
- Describe cross-module data flows, dependencies, and integration points
- Use arrow notation: `Component A -> Component B`
- Write in Chinese, but keep component names and technical terms in English

## Prohibited Content
- Do NOT include suggestions, evaluations, or future work
- Only document what exists today
- Stay objective and factual
